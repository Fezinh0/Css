Alinhamento de elementos e conteeúdo usando Flexbox

Flexbox é um mecanismo do CSS3 que permite aplicar facilmente alinhamento e distribuição de elementos e conteúdos no HTML.

Na prática, é uma técnica diferente para solucionar problemas e minimizar o grande uso de paddings, margins, floats, positions na resolução de dificuldades na construção de layouts.

As propriedades do flexbox são as seguintes

  • Flex-direction: Define qual direção o container deseja empilhar os itens (column, column-reverse, row, row-reverse).
  • Flex-wrap: Especifica se o itens flexíveis devem ser encapsulados ou não (flex-wrap, wrap, nowrap, wrap-reverse).
  • flex-flow: propriedade abreviada para definir Flex-direction e flex-wrap (flex-flow: row wrap).
  • Justify-content: Alinhamento de itens (Justify-content, center, flex-start, flex-end, space-around, space-between).
  • Align-items: propriedade é usada para alinhar os itens flexíveis
  • Align-content: Propriedade é usada para alinhar as linhas flexíveis
